# Mock Government Website - Educational Project

A professional-looking government website for educational purposes, showcasing the Democratic Republic of Testland.

## 🏛️ Features

- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Professional Styling**: Government-style color scheme (dark blue and gold)
- **Multiple Pages**:
  - Home: Landing page with announcements and departments
  - About Us: Government structure and core values
  - Services: Government services overview
  - News: Latest government announcements
  - Contact: Contact form and information

## 📄 Pages Included

1. **index.html** - Home page with hero section, announcements, and department overview
2. **about.html** - Information about the government, mission, and structure
3. **services.html** - Government services and programs
4. **news.html** - News and announcements from the government
5. **contact.html** - Contact form and government contact information
6. **styles.css** - All styling for the website

## 🎨 Color Scheme

- Primary Blue: #1a3a52
- Dark Blue: #0f2438
- Gold Accent: #d4af37
- Light Background: #f4f4f4

## 🚀 Usage

1. Save all files in the same directory
2. Open `index.html` in your web browser
3. Navigate through the pages using the navigation menu

## 📚 Educational Purpose

This website is designed for educational purposes to demonstrate:
- HTML structure and semantic markup
- CSS styling and responsive design
- Website navigation and user experience
- Professional web design principles
- Government website layout conventions
